About The Maze

The Maze is a 3D maze exploration game developed in C using the SDL2 library. It was created as a portfolio project for Holberton School.

Development was performed using Ubuntu 14.04 LTS with gcc 4.8.4.

Inspiration: This project was inspired by a desire to explore the capabilities of ray casting in game development and to create an engaging, immersive experience for users.
